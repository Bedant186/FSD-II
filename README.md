## Description
This project demonstrates frontend development using **React** and **Vite** through two structured experiments:

- **Experiment-1** focuses on building a basic Single Page Application with routing and multiple pages.

The application follows a component-based architecture and enables smooth navigation without page reloads.

## Experiment-1: Basic SPA Development

### Aim
To create a basic Single Page Application using React and Vite.

### Key Features
- Client-side routing using React Router
- Navigation without page reload
- Component-based UI
- Fast development using Vite

---

## Experiment-2: UI Design Components

### Aim
To design reusable UI components and integrate them into existing SPA pages.

### Tasks Performed
1. Created **5 UI design components**:
   - Button
   - TextField
   - Select
   - Rating
   - Checkbox

2. Copied **3 components/pages from Experiment-1**:
   - Home
   - About
   - Contact

3. Integrated **all 5 UI components** into the copied pages:
   - Forms enhanced using TextField, Select, and Checkbox
   - Actions handled using Button component
   - User interaction improved using Rating component
   - Uniform and reusable UI across pages
